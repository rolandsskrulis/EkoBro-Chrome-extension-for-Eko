

# EkoBro - Trusty bro in making a reviewers life a little bit easier, in Chrome extension form.


## Features
1. **Create Ticket Modal**
    - Adds a "create ticket" modal under questions on the main page.
    - Displays all created tickets directly under the modal.
2. **Super Expand Button**
    - Expands product videos even more for detailed viewing.
3. **Section Video Navigation**
    - Switch between section videos using arrow keys.
4. **Keyboard Shortcuts**
    - Add section to description: `Option + A`
    - Navigate sections: `Arrow up/down`
    - Open PDP: `Cmd + Shift + 7`
    - Open Backstage: `Cmd + Shift + 8`
    - Super Expand: `Cmd + Shift + 0`

## Installation
1. Download the zip archive from GitHub.
2. Unzip the archive.
3. Open Chrome and navigate to `chrome://extensions/`.
4. Enable **Developer mode** in the top right corner.
5. Click **Load unpacked** in the top left corner.
6. Select the unzipped folder of the extension.
